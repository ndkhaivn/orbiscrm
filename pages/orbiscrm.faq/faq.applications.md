                </a>

        </div>
        <div id="how-do-i-add-edit-the-reference-number-for-an-application" class="panel-collapse collapse noCrossRef">
            <div class="panel-body">
                <ol>
                    <li>Navigate to the specific <b>Application</b> > <b>Reference No</b>
                    </li>
                    <li>Click the【<b>Edit</b>】<img src="images/applications/overview/edit_btn.png" style="width: 4%"> button
                    </li>
                    <li>Type in the <b>Reference Number</b> from immiAccount
                    </li>
                    <li>Click the【<b>Save</b>】button
                    </li>
                </ol>
            </div>
        </div>
    </div>
    <!-- /.panel -->
    <div class="panel panel-default">
        <div class="panel-heading">
            <h4 class="panel-title">
                <a class="noCrossRef accordion-toggle" data-toggle="collapse" data-parent="#accordion" href="#how-do-i-set-up-a-reminder-for-a-progress-task-in-an-application">
                    How do I link the application to an existing folder in Dropbox?
                </a>
            </h4>
        </div>
        <div id="how-do-i-set-up-a-reminder-for-a-progress-task-in-an-application" class="panel-collapse collapse noCrossRef">
            <div class="panel-body">
                <ol>
                    <li>Navigate to the specific <b>Application</b> > <b>Application Folder</b>
                    </li>
                    <li>Click the【<b>Link</b>】<img src="images/link_folder_btn.png" style="width: 5%"> button
                    </li>
                    <li>Navigate to and select the folder that you want to link in the pop-up Dropbox window
                    </li>
                    <li>Click the【<b>Choose</b>】button
                    </li>
                    <li>Click the【<b>Confirm</b>】button in the pop-up Confirmation window
                    </li>
                </ol>
            </div>
        </div>
    </div>
    <!-- /.panel -->
    <div class="panel panel-default">
        <div class="panel-heading">
            <h4 class="panel-title">
                <a class="noCrossRef accordion-toggle" data-toggle="collapse" data-parent="#accordion" href="#why-the-linked-folder-disappears">
                    Why the linked folder disappears when I add/edit an immiAccount reference number to an application?
                </a>
            </h4>
        </div>
        <div id="why-the-linked-folder-disappears" class="panel-collapse collapse noCrossRef">
            <div class="panel-body">
                <ul>
                    <li>The application will be recognized as a <b>new application</b> when you add/edit an immiAccount reference number to it</li>
                    <li>If you want to work in the <b>same folder</b> as before, just link the application to it</li>
                    <li><b>OR</b> if you want to work in a <b>new folder</b>, create a new folder in Dropbox, and link the application to it</li>
                </ul>
            </div>
        </div>
    </div>
    <!-- /.panel -->
    <div class="panel panel-default">

                    How do I change the status for an application?
                </a>
            </h4>
        </div>
        <div id="how-do-i-change-the-status-for-an-application" class="panel-collapse collapse noCrossRef">
            <div class="panel-body">
                <ol>
                    <li>Navigate to the specific <b>Application</b> > <b>Status</b> <span>(A top-right button in the page)</span>
                    </li>
                    <li>Click the【<b>Status</b>】button
                    </li>
                    <li>Change the current <b>Application Status</b> in the pop-up window, including:
                        <ul>
                            <li>In Progress</li>
                            <li>Lodge</li>
                            <li>Finalised</li>
                        </ul>
                    </li>
                    <li>Click the【<b>Save</b>】button
                    </li>
                </ol>
            </div>
